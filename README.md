# Galaxy-ML-Robustness

Analyzing noise and blur robustness in weak-lensing galaxy image analysis using ML techniques.

## Project Overview
This project investigates how various noise levels and atmospheric blur affect the accuracy of galaxy morphological classification. We use deep learning models and astronomical archival data.

## Updates - 2026-04-27
- Investigated the impact of cosmological parameters on weak-lensing shear measurements.
- Integrated FLRW metric considerations for distance-dependent noise modeling.
